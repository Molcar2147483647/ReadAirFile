<br>
#基本仕様<br>
<br>
MUGENキャラ制作で使用されるSAEのAirファイルを読み取ります<br>
インクルードファイルとして使用<br>
<br>
<br>
#機能一覧<br>
<br>
対象のAirファイルを読み込み<br>
　対象ファイルの名前と拡張子を指定<br>
　ReadAirFile側のデータ格納先に対象ファイルのアニメーションデータが格納される<br>
　もしパラメータにエラーがあればエラー情報が表示されます<br>
<br>
読み込んだアニメションデータのパラメータの取得<br>
　ReadAirFile::関数名() でパラメータの取得が可能です<br>
<br>
<br>
#使用可能な関数一覧<br>
<br>
Airファイル読み込み方法<br>
　bool LoadAirFile("XXX.air") // 対象ファイルを読み込み(true:読み込み成功, false:読み込み失敗)<br>
<br>
各アニメ番号へのアクセスで使用する関数<br>
　参照 Anim(アニメ番号).XXX()　　　　　// アニメ番号の情報<br>
　参照 Anim(アニメ番号, 枚数番号).XXX() // アニメ番号と枚数番目の情報<br>
<br>
Anim(アニメ番号).XXX()でアクセス可能な関数<br>
　参照 Elem(枚数番号) // 枚数番号の情報の参照<br>
　int ID()　　　　　　 // 現在参照中のアニメ番号(0～2147483647)<br>
　bool Loop()　　　　 // ループ設定検知(false:ループ設定なし, true:ループ設定有り)<br>
　int LoopElem()　　　// ループ始点Elem(0～2147483647)<br>
<br>
Anim(アニメ番号, 枚数番号).XXX()でアクセス可能な関数<br>
　int SpriteID()　　 　// 画像識別番号xxx, YYYのxxxの部分(0～32767)<br>
　int SpriteNumber() // 画像識別番号xxx, YYYのYYYの部分(0～32767)<br>
　double PosX()　　 // 座標X(-2147483648～2147483647)<br>
　double PosY()　　 // 座標Y(-2147483648～2147483647)<br>
　int Time()　　　　  // Elem表示時間(-2147483648～2147483647)<br>
　bool Facing()　　　// 左右向き反転(反転なし:false, 判定有り:true)<br>
　bool VFacing()　　 // 上下向き反転(反転なし:false, 判定有り:true)<br>
　int AlphaA()　　　 // 透過設定Aの値(0～255)<br>
　int AlphaS()　　　 // 透過設定Sの値(0～255)<br>
　int AlphaD()　　　 // 透過設定Dの値(0～255)<br>
<br>
<br>
#エラー情報一覧<br>
<br>
アニメ番号の値が範囲外の場合<br>
　アニメーション番号が登録可能な数値の範囲(0～2147483647)を超えています<br>
<br>
アニメ番号が重複して登録されている場合<br>
　アニメーションが重複して登録されています<br>
<br>
空アニメを検知した場合<br>
　アニメーション内容が登録されていません<br>
　<br>
パラメータの値が範囲外の場合<br>
　SpriteIDが登録可能な数値の範囲(0～32767)を超えています<br>
　SpriteNumberが登録可能な数値の範囲(0～32767)を超えています<br>
　X座標が登録可能な数値の範囲(-2147483648～2147483647)を超えています<br>
　Y座標が登録可能な数値の範囲(-2147483648～2147483647)を超えています<br>
　アニメタイマーが登録可能な数値の範囲(-2147483648～2147483647)を超えています<br>
　AlphaAが登録可能な数値の範囲(0～255)を超えています<br>
　AlphaSが登録可能な数値の範囲(0～255)を超えています<br>
　AlphaDが登録可能な数値の範囲(0～255)を超えています<br>
<br>
#注意事項<br>
<br>
存在しないアニメ番号や枚数番号を指定すると例外がスローされます<br>
パラメータのエラー検知時、対象パラメータの値は初期値の0になります<br>
<br>
<br>
#使用例<br>
<br>
// アニメ5300番, 0Elem目のSpriteIDを取得<br>
ReadAirFile::Anim(5300, 0).SpriteID();<br>
<br>
// アニメ5300番のLoopstart設定を取得<br>
ReadAirFile::Anim(5300).Loop();<br>
