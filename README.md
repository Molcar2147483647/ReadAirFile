# ReadAirFile
## 基本仕様
MUGENキャラ制作で使用されるSAEのAirファイルを読み取り、各アニメーションのパラメータを取得できます  
アニメーションの組み立てをSAEで行い、それぞれのプログラムでAirファイルを読み込むことでパラメータを取り込むことが可能になります
  
  
## 機能一覧
##### 対象のAirファイルを読み込み
　対象ファイルの名前と拡張子を指定  
　ReadAirFile側のデータ格納先に対象ファイルのアニメーションデータが格納される  
　もしパラメータにエラーがあればエラー情報が表示されます  
  
##### 読み込んだアニメションデータのパラメータの取得
　ReadAirFile::関数名() でパラメータの取得が可能です  
  
  
## 使用可能な関数一覧
##### Airファイル読み込み方法  
　bool LoadAirFile("XXX.air") // 対象ファイルを読み込み(true:読み込み成功, false:読み込み失敗)  
  
##### 各アニメ番号へのアクセスで使用する関数
　参照 Anim(アニメ番号).XXX()　　　　　// アニメ番号の情報  
　参照 Anim(アニメ番号, 枚数番号).XXX() // アニメ番号と枚数番目の情報  
  
##### Anim(アニメ番号).XXX()でアクセス可能な関数
　参照 Elem(枚数番号) // 枚数番号の情報の参照  
　int ID()　　　　　　 // 現在参照中のアニメ番号(0～2147483647)  
　bool Loop()　　　　 // ループ設定検知(false:ループ設定なし, true:ループ設定有り)  
　int LoopElem()　　　// ループ始点Elem(0～2147483647)  
  
##### Anim(アニメ番号, 枚数番号).XXX()でアクセス可能な関数
　int SpriteID()　　 　// 画像識別番号xxx, YYYのxxxの部分(0～32767)  
　int SpriteNumber() // 画像識別番号xxx, YYYのYYYの部分(0～32767)  
　double PosX()　　 // 座標X(-2147483648～2147483647)  
　double PosY()　　 // 座標Y(-2147483648～2147483647)  
　int Time()　　　　  // Elem表示時間(-2147483648～2147483647)  
　bool Facing()　　　// 左右向き反転(反転なし:false, 判定有り:true)  
　bool VFacing()　　 // 上下向き反転(反転なし:false, 判定有り:true)  
　int AlphaA()　　　 // 透過設定Aの値(0～255)  
　int AlphaS()　　　 // 透過設定Sの値(0～255)  
　int AlphaD()　　　 // 透過設定Dの値(0～255)  
  
  
## エラー情報一覧
##### アニメ番号の値が範囲外の場合
　アニメーション番号が登録可能な数値の範囲(0～2147483647)を超えています  
  
##### アニメ番号が重複して登録されている場合
　アニメーションが重複して登録されています  
  
##### 空アニメを検知した場合
　アニメーション内容が登録されていません  
　  
##### パラメータの値が範囲外の場合
　SpriteIDが登録可能な数値の範囲(0～32767)を超えています  
　SpriteNumberが登録可能な数値の範囲(0～32767)を超えています  
　X座標が登録可能な数値の範囲(-2147483648～2147483647)を超えています  
　Y座標が登録可能な数値の範囲(-2147483648～2147483647)を超えています  
　アニメタイマーが登録可能な数値の範囲(-2147483648～2147483647)を超えています  
　AlphaAが登録可能な数値の範囲(0～255)を超えています  
　AlphaSが登録可能な数値の範囲(0～255)を超えています  
　AlphaDが登録可能な数値の範囲(0～255)を超えています  
  
  
## 注意事項
存在しないアニメ番号や枚数番号を指定すると例外がスローされます  
パラメータのエラー検知時、対象パラメータの値は初期値の0になります  
  
  
## 使用例
// アニメ5300番, 0Elem目のSpriteIDを取得  
ReadAirFile::Anim(5300, 0).SpriteID();  
  
// アニメ5300番のLoopstart設定を取得  
ReadAirFile::Anim(5300).Loop();
